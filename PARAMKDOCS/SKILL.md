# Project Analyzer Skill

This skill performs comprehensive analysis of software projects including:
- Project structure and architecture
- Technology stack identification
- Code quality assessment
- Documentation generation
- User and developer guide creation
- MKdocs preparation

## Analysis Components

1. **Project Overview**: Name, purpose, business domain
2. **Technical Stack**: Languages, frameworks, libraries, tools
3. **Architecture**: Patterns, design decisions, components
4. **Functionality**: Key features, user workflows
5. **Development**: Build process, testing, deployment
6. **Documentation**: Current state and gaps
7. **Recommendations**: Improvements, optimizations

## Output Format

Generates comprehensive markdown documentation suitable for:
- Developer onboarding
- User guides
- Technical specifications
- MKdocs integration
- Project presentations
